<h2>Lesson 1: Introduction to CSS</h2>
<div class="section">
  <p><strong>What is CSS?</strong><br>
  CSS stands for <strong>Cascading Style Sheets</strong>. It defines how HTML elements are displayed on screens and helps separate content from design. It makes websites easier to maintain and style consistently.</p>

  <p><strong>Why Use CSS?</strong></p>
  <ul>
    <li>Controls layout, colors, and fonts</li>
    <li>Keeps design consistent across pages</li>
    <li>Saves time with reusable styles</li>
    <li>Makes sites responsive and accessible</li>
  </ul>

  <p><strong>Layers of a Webpage:</strong></p>
  <ul>
    <li><strong>Content Layer (HTML):</strong> Structure and information.</li>
    <li><strong>Presentation Layer (CSS):</strong> Design and visual style.</li>
    <li><strong>Behavior Layer (JavaScript):</strong> Interactivity and user actions.</li>
  </ul>

  <p><strong>Benefits of CSS:</strong></p>
  <ul>
    <li>Saves time and reduces repetitive coding</li>
    <li>Easy maintenance and updates</li>
    <li>Improves website appearance and layout</li>
  </ul>

  <p><strong>Capabilities of CSS:</strong></p>
  <ul>
    <li>Update layouts quickly</li>
    <li>Position and layer elements</li>
    <li>Create custom tags and effects</li>
  </ul>

  <p><strong>Advantages of CSS:</strong></p>
  <ul>
    <li>Faster loading and easier updates</li>
    <li>Reusable for multiple pages</li>
    <li>Flexible and user-friendly design</li>
  </ul>

  <p><strong>Prerequisite:</strong> Basic understanding of HTML or XHTML before learning CSS.</p>
</div>

<!-- Lesson 2 -->
<h2>Lesson 2: Inline, Internal, and External Style Sheets</h2>

<div class="section">
  <p><strong>What are the Three Types of CSS?</strong><br>
  CSS can be added to HTML in three main ways: Inline, Internal (or Embedded), and External. Each type has its own purpose and level of convenience when styling web pages.</p>

  <p><strong>1. Inline Style Sheet</strong><br>
  Inline CSS is written directly within an HTML tag using the <code>style</code> attribute. It affects only that specific element. While it’s quick for small edits, it’s not recommended for large websites because it can become repetitive and hard to manage.</p>

  <p><strong>Example:</strong>  
  <code>&lt;p style="color: pink; text-align: center;"&gt;Hello World&lt;/p&gt;</code></p>

  <p><strong>2. Internal (Embedded) Style Sheet</strong><br>
  Internal CSS is written inside the <code>&lt;style&gt;</code> tag within the HTML document’s <code>&lt;head&gt;</code> section. It applies to the entire page and is easier to manage than inline CSS, but changes only affect that one file.</p>

  <p><strong>Example:</strong><br>
  <code>&lt;style&gt; p { color: blue; } &lt;/style&gt;</code></p>

  <p><strong>3. External Style Sheet</strong><br>
  External CSS is written in a separate <code>.css</code> file and linked to the HTML using the <code>&lt;link&gt;</code> tag. This is the most efficient method for large or multi-page websites because one file can control the style for all pages.</p>

  <p><strong>Example:</strong><br>
  <code>&lt;link rel="stylesheet" href="style.css"&gt;</code></p>

  <p><strong>Advantages of Each Type:</strong></p>
  <ul>
    <li><strong>Inline:</strong> Quick and specific to one element.</li>
    <li><strong>Internal:</strong> Easy to apply styles for a single page.</li>
    <li><strong>External:</strong> Best for consistency and reusability across multiple pages.</li>
  </ul>

  <p><strong>Summary:</strong><br>
  Inline CSS is best for one-time styling, Internal CSS for small projects, and External CSS for large or multi-page websites. External style sheets are preferred in real-world web design because they save time and keep code clean.</p>
</div>
